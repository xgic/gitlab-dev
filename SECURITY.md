# Security

Report vulnerabilities privately to the XGIC maintainers via the GitHub Security advisory workflow for this repository when available, or through the contact listed on https://github.com/xgic.

This repository must not contain private infrastructure hosts, production project paths, credentials, or private tracker identifiers. See [BASE-STANDARDS](https://github.com/xgic/ai/blob/main/docs/BASE-STANDARDS-FOR-ORCHESTRATED-REPOS.md).
